# HexletCode

This is simple form generator, created as study project

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'hexlet_code'
```

And then execute:

    $ bundle install

Or install it yourself as:

    $ gem install hexlet_code

## Development

After checking out the repo, run `make install`
